Das Microsoft Ajax Content Delivery Network (CDN) hostet gängige JavaScript-Bibliotheken von Drittanbietern wie jQuery und ermöglicht ihnen das einfache Hinzufügen zu Ihren Webanwendungen. Sie können z. B. mit jQuery beginnen, das &lt;auf&gt; diesem CDN gehostet wird, indem Sie einfach ein Skript-Tag zu Ihrer Seite hinzufügen, das auf ajax.aspnetcdn.com verweist.

Durch die Nutzung des CDN können Sie die Leistung Ihrer Ajax-Anwendungen deutlich verbessern. Der Inhalt des CDN wird auf Servern auf der ganzen Welt zwischengespeichert. Darüber hinaus ermöglicht das CDN Browsern die Wiederverwendung zwischengespeicherter JavaScript-Dateien von Drittanbietern für Websites, die sich in verschiedenen Domänen befinden.

Das CDN unterstützt SSL (HTTPS), falls Sie eine Webseite mit der Layer secure Sockets bereitstellen müssen.

Das CDN hostet die folgenden Skriptbibliotheken von Drittanbietern, die von den Besitzern dieser Bibliotheken hochgeladen wurden und für Sie lizenziert sind:

- jQuery (www.jquery.com)
- jQuery-Benutzeroberfläche (www.jqueryui.com)
- jQuery Mobile (www.jquerymobile.com)
- jQuery-Validierung (https://jqueryvalidation.org/)
- jQuery-Zyklus (www.malsup.com/jquery/cycle/)
- jQuery DataTables (http://datatables.net/)

Das Microsoft Ajax CDN enthält auch die folgenden Bibliotheken, die von Microsoft hochgeladen wurden:

- ASP.NET AJAX
- ASP.NET MVC-JavaScript-Dateien
- ASP.NET SignalR JavaScript-Dateien

Microsoft beansprucht keinen Besitz an Bibliotheken von Drittanbietern, die auf diesem CDN gehostet werden. Die Urheberrechtsinhaber der Bibliotheken lizenzieren Ihnen diese Bibliotheken. Alle Rechte, die Sie zum Herunterladen und Nutzen solcher Bibliotheken haben, werden ausschließlich von den jeweiligen Urheberrechtsinhabern gewährt. Da es sich nicht um Microsoft-Bibliotheken handelt, stellt Microsoft keine Garantien oder Lizenzen für geistige Eigentumsrechte (einschließlich stillschweigender Patentrechte) für die auf diesem CDN gehosteten Bibliotheken von Drittanbietern bereit.

Wenn Sie Ihre JavaScript-Bibliothek einreichen möchten und Ihre Bibliothek eine der http://trends.builtwith.com) besten JavaScript-Bibliotheken ist (wie auf oder Erweiterungen/Plugins zu diesen Bibliotheken aufgeführt, die (a) beliebt sind; oder (b) hilfreich für die Verwendung auf ASP.NET dann wenden Sie sich bitte an AjaxCDNSubmission@Microsoft.com.

## <a name="ajaxmicrosoftcom-renamed-to-ajaxaspnetcdncom"></a>ajax.microsoft.com in ajax.aspnetcdn.com umbenannt

Das CDN verwendet den microsoft.com Domänennamen und wurde geändert, um den aspnetcdn.com Domänennamen zu verwenden. Diese Änderung wurde vorgenommen, um die Leistung zu erhöhen, da ein Browser, der auf die microsoft.com Domäne verweist, mit jeder Anforderung Cookies aus dieser Domäne über den Draht senden würde. Durch das Umbenennen in einen anderen Domänennamen als microsoft.com Leistung um bis zu 25 % gesteigert werden. Hinweisajax.microsoft.com weiterhin funktioniert, aber ajax.aspnetcdn.com wird empfohlen.

- Altes Format:https://ajax.microsoft.com/ajax/jQuery/jquery-1.8.0.js
- Neues Format:https://ajax.aspnetcdn.com/ajax/jQuery/jquery-1.8.0.js
