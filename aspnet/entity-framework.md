# <a name="entity-framework"></a>Entity Framework

Entity Framework (EF) ist eine Objekt relationale Zuordnung, die .NET-Entwicklern die Arbeit mit relationalen Daten mithilfe von domänenspezifischen Objekten ermöglicht. Es entfällt die Notwendigkeit für den Großteil des Datenzugriffs-Codes, den Entwickler normalerweise schreiben müssen.
