<span data-ttu-id="d9b63-163">Bevor wir zu einer Datenbank gehen und über Modelle sprechen, sollten wir uns jedoch zunächst mit der Übergabe von Informationen vom Controller an eine Ansicht beschäftigen.</span><span class="sxs-lookup"><span data-stu-id="d9b63-163">Before we go to a database and talk about models, though, let's first talk about passing information from the Controller to a View.</span></span> <span data-ttu-id="d9b63-164">Wir möchten eine Ansichts Vorlage übergeben, um eine HTML-Antwort an einen Client zu rendereinigen.</span><span class="sxs-lookup"><span data-stu-id="d9b63-164">We want to pass what a view template requires in order to render an HTML response to a client.</span></span> <span data-ttu-id="d9b63-165">Diese Objekte werden in der Regel erstellt und von einer Controller Klasse an eine Ansichts Vorlage weitergegeben.
