## <a name="avoiding-cross-site-scripting-xss-attacks"></a><span data-ttu-id="4012b-152">Vermeiden von Cross-Site Scripting (XSS)-Angriffen</span><span class="sxs-lookup"><span data-stu-id="4012b-152">Avoiding Cross-Site Scripting (XSS) Attacks</span></span>

<span data-ttu-id="4012b-153">Wenn Sie Eingaben von einem Benutzer akzeptieren und diese Eingabe auf Ihrer Website erneut anzeigen, können Sie Ihre Website möglicherweise für Cross-Site Scripting (XSS)-Angriffe öffnen.</span><span class="sxs-lookup"><span data-stu-id="4012b-153">Whenever you accept input from a user, and redisplay that input on your website, you potentially open your website to Cross-Site Scripting (XSS) Attacks.</span></span> <span data-ttu-id="4012b-154">Theoretisch könnte ein böswilliger Hacker JavaScript-Code übermitteln, der ausgeführt wird, wenn die Eingabe erneut angezeigt wird.</span><span class="sxs-lookup"><span data-stu-id="4012b-154">In theory, a malicious hacker could submit JavaScript code that gets executed when the input is redisplayed.</span></span> <span data-ttu-id="4012b-155">Das JavaScript könnte verwendet werden, um Benutzer Kennwörter oder andere vertrauliche Informationen zu stehlen.</span><span class="sxs-lookup"><span data-stu-id="4012b-155">The JavaScript could be used to steal user passwords or other sensitive information.</span></span>

<span data-ttu-id="4012b-156">Normalerweise können Sie XSS-Angriffe durch HTML-Codierung von allen Eingaben, die Sie von einem Benutzer abrufen, übertragen, bevor Sie ihn auf einer Webseite anzeigen.</span><span class="sxs-lookup"><span data-stu-id="4012b-156">Normally, you can defeat XSS attacks by HTML encoding whatever input you retrieve from a user before displaying it in a web page.</span></span> <span data-ttu-id="4012b-157">Die HTML-Codierung der Ausgabe des HTML-Editors würde jedoch nicht nur &lt;Skript&gt; Tags codieren, sondern auch alle HTML-Tags codieren.</span><span class="sxs-lookup"><span data-stu-id="4012b-157">However, HTML encoding the output of the HTML Editor would not only encode &lt;script&gt; tags, it would also encode all HTML tags.</span></span> <span data-ttu-id="4012b-158">Dies bedeutet, dass Sie die gesamte Formatierung verlieren würden, z. b. Schriftart, Schrift Grad und Hintergrundfarbe.</span><span class="sxs-lookup"><span data-stu-id="4012b-158">In other words, you would lose all of the formatting such as the font type, font size, and background color.</span></span>

<span data-ttu-id="4012b-159">Wenn Sie vertrauliche Informationen von Benutzern erfassen, z. b.
