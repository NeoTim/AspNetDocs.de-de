# <a name="how-do-i-use-the-aspnet-ajax-alwaysvisible-control-extender"></a>[Gewusst wie:] Verwenden Sie den ASP.NET AJAX alwaysvisible-Steuerelement-Extender?

von [Joe Stagner](https://github.com/JoeStagner)

Sehen Sie, wie einfach es ist, ein Seitenelement an einer Position auf dem Bildschirm zu verankern, wo es immer sichtbar bleibt, unabhängig davon, wie weit der Benutzer das Dokument durchblättern kann. Das Floating Page-Element kann aus statischem Text und einem AJAX-basierten, kontextabhängigen Hilfefeld bestehen.
