# <a name="the-datapager-control"></a>Das DataPager-Steuerelement

von [Joe Stagner](https://github.com/JoeStagner)

Im Allgemeinen wird das DataPager-Steuerelement mit dem ListView-Steuerelement verwendet. Das ListView-Steuerelement ist Daten gesteuert. Erstellen Sie zunächst ein ListView-Steuerelement, und stellen Sie ihm einige Daten bereit. Legen Sie als nächstes das DataPager-Steuerelement auf der Seite ab, um die Funktionalität der Webseite zu optimieren. Wenn Sie das ListView-Steuerelement nicht Gewusst wie: Video ansehen, wäre es am besten, es vor der Verwendung des DataPager-Steuer Elements zu überprüfen.
