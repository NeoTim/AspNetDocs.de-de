# <a name="how-do-i-associate-ajax-client-behavior-with-an-aspnet-server-control"></a>[Gewusst wie:] Zuordnen des AJAX-Client Verhaltens zu einem ASP.NET-Server Steuerelement

von [Chris Pels](https://twitter.com/chrispels)

Erfahren Sie, wie Sie ein Client Verhalten auf der Grundlage von Microsoft AJAX erstellen, das mit einem ASP.NET-Server Steuerelement auf einer ASPX-Seite verknüpft werden kann. Das Client Verhalten erweitert die Funktionalität des standardmäßigen ASP.net-Steuer Elements durch die Zuordnung, anstatt ein neues benutzerdefiniertes Steuerelement zu erstellen. Daher kann das Verhalten je nach Implementierung über mehrere Instanzen oder Typen von ASP.NET-Steuerelementen wieder verwendet werden.
