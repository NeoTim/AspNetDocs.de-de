# <a name="how-do-i-use-the-conditional-updatemode-of-the-updatepanel"></a>[Gewusst wie:] Verwenden der bedingten UpdateMode von UpdatePanel?

durch [Joe Stagner](https://github.com/JoeStagner)

Das UpdatePanel von ASP.NET AJAX umfasst eine UpdateMode-Eigenschaft, die auf "Always" oder "Bedingt" festgelegt werden kann. Der Standardwert ist immer in diesem Fall das UpdatePanel stets den Inhalt während eines asynchronen Postbacks aktualisiert wird. In diesem Video erfahren wir, wie wir die UpdateMode auf bedingte, festlegen können in der Fall UpdatePanel nur den Inhalt aktualisiert wird, sobald unsere serverseitigen Code die Update-Methode aufruft. Dadurch können Sie bedingten Logik in Ihrem C#- oder Visual Basic-Code verwenden, um festzustellen, ob das UpdatePanel seinen Inhalt während des aktuellen asynchronen Postbacks aktualisiert wird.
