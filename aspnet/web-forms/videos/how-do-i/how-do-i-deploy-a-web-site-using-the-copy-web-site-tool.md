# <a name="how-do-i-deploy-a-web-site-using-the-copy-web-site-tool"></a>[Gewusst wie:] Bereitstellen einer Website mit dem Tool zum Kopieren von Websites

von [Chris Pels](https://twitter.com/chrispels)

In diesem Video zeigt Chris Pels, wie eine Website mit dem Tool zum Kopieren von Websites in Visual Studio bereitgestellt wird. Zunächst erfahren Sie, wie Sie eine Verbindung mit einer Remote Website herstellen und eine gesamte Website am Ziel Speicherort bereitstellen. Als Nächstes erfahren Sie, wie Sie eine gesamte Quell-und Zielwebsite synchronisieren. Weitere Informationen finden Sie unter Kopieren und synchronisieren einzelner Dateien oder Gruppen von Dateien, einschließlich der Art und Weise, wie das Tool zum Kopieren von Websites Status Indikatoren von Unterschieden zwischen Dateien bereitstellt. Im Rahmen dieses Vorgangs sehen Sie, wie das Tool Optionen auf Datei Basis zur Entscheidungsfindung bereitstellt, wenn Unterschiede im Status zwischen zwei oder mehr Dateien bestehen.
