# <a name="how-do-i-enforce-coding-standards-with-code-analysis"></a>[Gewusst wie:] Erzwingen von Codierungs Standards mit der Code Analyse

von [Chris menegay](https://twitter.com/CMenegay)

Mit Visual Studio Team System können Sie die Code Analyse Regeln, die für Klassen Bibliotheks Projekte, Website Projekte und die Quell Code Verwaltung gelten, genau steuern. In diesem Video erfahren Sie, wie Sie die Fehler und Warnungen angeben, die bei Projektbuilds und Code Eincheck Vorgängen angewendet werden, und es wird beschrieben, wie Sie eine bestimmte Regel von der Anwendung auf einen bestimmten Klassenmember unterdrücken. Mit dieser Konfigurierbarkeit können Sie Codierungsstandards erzwingen, um die Qualität Ihres Codes zu verbessern, und gleichzeitig sicherstellen, dass nicht anwendbare Regeln nie auf die gleiche Weise werden.
