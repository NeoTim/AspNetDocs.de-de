# <a name="how-do-i-set-up-distributed-load-testing-for-high-volume-tests"></a>[Gewusst wie:] Einrichten verteilter Auslastungs Tests für Tests mit hohem Volumen

von [Chris menegay](https://twitter.com/CMenegay)

Mithilfe von Visual Studio Team System können Sie verteilte Auslastungs Tests einrichten, indem Sie ein Test-"Rig" erstellen, das einen Controller auf einem Remote Computer und mehrere Agents auf denselben oder zusätzlichen Remote Computern umfasst. Dieses Video führt Sie durch die Installation des Controllers und der Agents und die anschließende Konfiguration Ihrer Testfälle, um Sie mithilfe des Remote Controllers auszuführen. Wie Sie sehen werden, vereinfacht Visual Studio Team System das Einrichten verteilter Auslastungs Tests für automatisierte Tests mit hohem Volumen.
