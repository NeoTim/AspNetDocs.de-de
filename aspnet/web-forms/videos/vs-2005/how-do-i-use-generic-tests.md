# <a name="how-do-i-use-generic-tests"></a>[Gewusst wie:] Generische Tests verwenden?

von [Chris menegay](https://twitter.com/CMenegay)

Bei einem generischen Test handelt es sich um ein vorhandenes Programm, das umschließt und als Test in Visual Studio Team System zur Verfügung gestellt wird. In diesem Video wird veranschaulicht, wie ein umschlendetes Shellprogramm einen Eingabe Wert aus der Testumgebung annimmt und wie der Ausgabewert des Programms verwendet wird, um zu bestimmen, ob der Test erfolgreich war oder fehlgeschlagen ist.
