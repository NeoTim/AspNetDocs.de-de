# <a name="the-effects-of-caching"></a>Die Auswirkungen der Zwischenspeicherung

von Eric Lee

Fast jede Webanwendung muss heute auf Daten aus einer Datenbank zugreifen, aber dieser häufige Vorgang kann sich auf die Gesamtleistung der Anwendung auswirken. Dieses Video veranschaulicht, wie Sie Visual Studio Team System dabei unterstützen können, zu ermitteln, ob Ihre Anwendung redundante Aufrufe an die Datenbank durchführt. Anschließend wird veranschaulicht, wie Sie mithilfe von ASP.NET 2,0 DataSource Caching-Funktionen redundante Aufrufe ausschließen können.
